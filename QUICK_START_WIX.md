# Quick Start: Adding Code to Wix - Step by Step

## ‚ö° Fastest Method: Copy Content Manually

### Step 1: Update Your Homepage (5 minutes)

1. **In Wix Editor, on your Homepage:**

   - **Delete the old "Create Studio" content:**
     - Click on the astronaut image ‚Üí Delete
     - Click on "Create Studio Pro" text ‚Üí Delete
     - Keep your header/navigation

2. **Add New Hero Section:**
   - Click **"Add"** button (left sidebar)
   - Click **"Strip"** ‚Üí Choose full-width strip
   - Click **"Add"** ‚Üí **"Heading"**
   - Type: **"ASPM Agent System"**
   - Click **"Add"** ‚Üí **"Text"**
   - Copy this text from `index.html`:
     ```
     A comprehensive multi-agent system for enterprise architecture, 
     change management, and strategic planning
     ```

3. **Style the Hero:**
   - Select the strip
   - In right panel, click **"Background"**
   - Choose **"Color"** ‚Üí Set to blue gradient
   - Or use **"Image"** if you prefer

### Step 2: Add Quick Start Cards (10 minutes)

1. **Add a new strip below hero**

2. **Add heading:** "Quick Start"

3. **Add 3 boxes/cards:**
   - Click **"Add"** ‚Üí **"Box"** (repeat 3 times)
   - In each box, add:
     - **Card 1:** Copy from `index.html` ‚Üí "üöÄ Dashboard (Streamlit)" section
     - **Card 2:** Copy from `index.html` ‚Üí "‚ö° API Server (FastAPI)" section  
     - **Card 3:** Copy from `index.html` ‚Üí "üìö Documentation" section

### Step 3: Update Navigation Menu (2 minutes)

1. **Click on your menu/navigation element**

2. **Update menu items to:**
   - Home
   - About
   - Features
   - Agents
   - Documentation
   - Contact

3. **Remove old items:** "Reviews", "Top List", "Education", "Portfolio"

### Step 4: Create New Pages (15 minutes)

1. **Add Pages:**
   - Click **"Pages & Menu"** (left sidebar)
   - Click **"+ Add Page"** for each:
     - About
     - Features
     - Agents
     - Documentation
     - Contact

2. **Add Content to Each Page:**
   
   **For About Page:**
   - Open `about.html` in a text editor
   - Copy the text content (skip HTML tags like `<h1>`, `<p>`, etc.)
   - Paste into Wix text elements
   
   **For Features Page:**
   - Open `features.html`
   - Copy feature descriptions
   - Create cards/boxes for each feature
   
   **For Agents Page:**
   - Open `agents.html`
   - Copy each agent description
   - Create a grid of agent cards
   
   **For Documentation Page:**
   - Open `documentation.html`
   - Copy code examples and API info
   - Use Wix code blocks for code snippets
   
   **For Contact Page:**
   - Open `contact.html`
   - Add contact form: **"Add"** ‚Üí **"Contact"** ‚Üí **"Form"**
   - Copy contact information text

### Step 5: Apply Colors & Styling (5 minutes)

1. **Set Site Colors:**
   - Click **"Site Design"** (left sidebar)
   - Click **"Color Palette"**
   - Set **Primary Color:** `#2563eb` (blue)
   - Set **Accent Color:** `#0ea5e9` (light blue)

2. **Set Fonts:**
   - In **"Site Design"** ‚Üí **"Fonts"**
   - Choose **"Inter"** or **"Open Sans"**
   - Apply to all headings and text

3. **Style Buttons:**
   - Select any button
   - In right panel, set:
     - Background: Primary color
     - Border radius: 8px
     - Add shadow

### Step 6: Save & Publish

1. **Click "Save"** (top right)

2. **Click "Preview"** to check

3. **Click "Publish"** when ready!

## Important Notes

- **Wix doesn't use HTML directly** - copy the text content, not HTML tags
- **Use Wix elements** (text boxes, buttons, strips, cards) to build the layout
- **Wix automatically makes it mobile responsive** - no extra work needed
- **Save frequently** to avoid losing work
- **Preview before publishing** to catch any issues

### System Architecture
**Title:** System Architecture

**Description:** The ASPM Agent System is built on a modular, extensible architecture with clear separation of concerns.

**Architecture Components:**
1. **Agents** - Base agent framework with capability-based architecture. 10 specialized agents for different domains.
2. **Core Engine** - Context management, data models, and services layer. Provides foundation for all agents.
3. **Orchestrator** - Agent routing, strategy building, and OKR management. Coordinates multi-agent workflows.
4. **Personas** - Communication style customization and tone application. 9 specialized personas for different contexts.
5. **Safety Wrapper** - ASPM-Guardian provides file write protection, module mutation checks, and safety registry.

### Quick Start
**Title:** Quick Start

**Installation:**
```bash
# Install dependencies
pip install -r requirements.txt
```

**Start Dashboard:**
```bash
# Windows
start_dashboard.bat

# Or directly
streamlit run aspm_streamlit/app.py
```
Access at: `http://localhost:8501`

**Start API Server:**
```bash
# Windows
start_api_server.bat

# Or directly
python aspm\server_fastapi.py
```
Access API docs at: `http://localhost:8000/docs`

### API Endpoints
**Title:** API Endpoints

**1. PMO Value Ring Assessment**
- `POST /api/pmo/value_ring/assess`
- Assess PMO service catalog and calculate value scores across PMI governance domains.

**2. ADKAR Change Readiness**
- `POST /api/ocm/adkar/intake`
- Assess organizational change readiness using ADKAR methodology.

**3. SPM Pattern Detection**
- `POST /api/spm/patterns/detect`
- Detect SPM/PPM administration patterns and symptoms.

**4. OKR Composition**
- `POST /api/okr/compose`
- Generate OKRs from strategic goals.

**5. KPI Composition**
- `POST /api/kpi/compose`
- Generate KPIs and signals for performance measurement.
